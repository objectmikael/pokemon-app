# Pokemon App

## How the App Works

- The Hompepage for the app is [localhost:3005/pokemonListHomepage](localhost:3005/pokemonListHomepage)
    - This page contains the list of Pokemon in the database
- The "Generate Pokemon" button allows a user to randomly create a new pokemon form the PokeApi
- The "Search for Pokemon" button allows a user to lookup any pokemon from the PokeApi and show details of the pokemon on a seperate page
- The "View Your Favorite Pokemon" button allows a user to view their favorite pokemon, marked when isFavorite option is checked  
- The "Customize Your Own Pokemon" button allows a user to create a unique Pokemon and save to the database
